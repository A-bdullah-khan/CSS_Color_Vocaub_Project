# CSS_Color_Vocaub_Project
A simple Spanish vocabulary webpage built using HTML and CSS. This project displays different colors in Spanish along with their corresponding images and styled text

FEATURES:
1) Main heading and subheading for the vocabulary topic
2) Display of Spanish color names
3) Images representing each color
4) Colored text styling matching each color
5) Consistent image size using CSS

CONCEPTS USED:
1) HTML Document Structure
2) Headings and Image Tags
3) CSS Selectors
4) CSS Styling
5) Linking External CSS File
